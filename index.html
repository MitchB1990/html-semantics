Semantic Elements Summary

Semantic elements clearly describe their meaning and usage to both the browser and developer. These tags also help make a document/website more searchable and create a website structure that is more easily understood. These tags also are used by assistive devices and apps to help render/translate content in an accessible manner.

Examples of semantic element tags include:
  • header
  • footer
  • nav
  • aside
  • section
  • article
